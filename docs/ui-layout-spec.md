# 共通 UI レイアウト設計書

## 画面構成

```
┌──────────────────────────────────────────────────────────┐
│ ツールバー: [録音/停止] [キャンセル] [エクスポート] [設定] │
├──────────────────────────────────────────────────────────┤
│ ▼ 入力（折りたたみ可能）              [録音中/録画中]     │
│   音源選択 Picker + レベルメーター + 画面プレビュー       │
├──────────────────────────────────────────────────────────┤
│ ▼ リアルタイム文字起こし（折りたたみ可能）  [言語ラベル]  │
│   [文字起こしプレビュー+コピー] | [翻訳+言語+コピー]     │
├──────────────────────────────────────────────────────────┤
│ ▼ 音声文字起こし（折りたたみ可能）                       │
│   ファイル選択（点線枠 D&D）                              │
│   プレーヤー（常時表示）                                  │
│   [文字起こし結果+コピー]     | [翻訳+言語+コピー]       │
├──────────────────────────────────────────────────────────┤
│ ▼ 要約（折りたたみ可能）                                 │
│   [要約結果+コピー]           | [翻訳+言語+コピー]       │
├──────────────────────────────────────────────────────────┤
│ ステータスバー: CPU アプリ X% / 全体 X% | メモリ X MB     │
└──────────────────────────────────────────────────────────┘
```

## 仕様

- すべてのセクションが折りたたみ可能（▶/▼ アイコン）
- 折りたたみヘッダー: 大きめフォント + アクセントカラー背景
- ステータス情報（録音中、言語判定）はヘッダー内に右寄せ表示
- テキストエリア: ウィンドウサイズに自動フィット、スクロールバー表示
- 各テキストエリアにコピーボタン
- 翻訳パネル: 言語 Picker + コピーボタンが同一ライン
- ファイル選択: 点線枠（未選択時・選択後とも統一）
- 録音/停止/キャンセル: ツールバーに統合（音源選択に応じてアイコン変更）
- 設定でリアルタイム文字起こし無効時: セクション全体を非表示
- 録音/録画開始時: 全テキスト + ファイル選択をクリア
- 画面録画: 音源選択 Picker 内に統合（動画＋音声固定）

## OS 別の実装マッピング

| UI 要素 | macOS (SwiftUI) | Windows (WinUI 3) |
|---------|-----------------|-------------------|
| 折りたたみ | カスタム Button + if | Expander |
| 左右分割 | HStack + Divider | Grid (2列) |
| ファイル D&D | .onDrop | DragOver/Drop |
| Picker | Picker(.menu) | ComboBox |
| コピーボタン | NSPasteboard | Clipboard.SetContent |
| スクロール | ScrollView(showsIndicators) | ScrollViewer |
| ツールバー | .toolbar + ToolbarItem | CommandBar |
